# bootdev

Course projects from [boot.dev](https://boot.dev), consolidated into a monorepo.

Each directory is a self-contained project with its own `flake.nix`. Enter a dev shell with `nix develop`.

## Projects

| Dir | Course | Stack |
|-----|--------|-------|
| `bookbot` | Learn Python | Python |
| `worldbanc` | Learn Bash | Bash |
| `webflyx` | Learn Git | Markdown/Git |
| `webflyx-local` | Learn Git (local) | Markdown/Git |
| `git-2/megacorp` | Learn Git 2 | Git |
| `goserver` | Learn HTTP Servers | Go, Python |
| `chirpy` | Learn Web Servers | Go, PostgreSQL |
| `heifer` | Learn JavaScript | Node.js |
| `blog-aggregator` | Learn SQL | Go, PostgreSQL |
| `rabbit` | Learn Pub/Sub | Go, RabbitMQ |
| `godex` | Learn Go | Go |
| `agentpy` | Build an AI Agent | Python |
| `pysteroid` | Learn OOP | Python |
