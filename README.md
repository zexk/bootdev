# bootdev

Course projects from [boot.dev](https://boot.dev), consolidated into a monorepo.

Each directory is a self-contained project with its own `flake.nix`. Enter a dev shell with `nix develop`.

## Projects

| Dir | Course | Stack |
|-----|--------|-------|
| `bookbot` | Build a Bookbot | Python |
| `worldbanc` | Learn Linux | Bash |
| `webflyx` | Learn Git | Git |
| `webflyx-local` | Learn Git | Git |
| `git-2/megacorp` | Learn Git | Git |
| `pysteroid` | Build an Asteroids Game | Python |
| `agentpy` | Build an AI Agent | Python |
| `heifer` | Learn JavaScript | JavaScript |
| `godex` | Build a Pokedex | Go |
| `goserver` | Learn Docker | Go |
| `chirpy` | Learn HTTP Servers | Go, PostgreSQL |
| `blog-aggregator` | Build a Blog Aggregator | Go, PostgreSQL |
| `rabbit` | Learn Pub/Sub Architecture | Go, RabbitMQ |
